5.бһңбһ¶бһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸ Folderбҹ– migrations, seeders, models бһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ”
****Migration бһ”бҹ’бһҡбҹҖбһ”бһҠбһјбһ…бһҮбһ¶ "Version Control (бһҠбһјбһ… Git)" бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Database бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
рҹ‘үбһ”бһүбҹ’бһ бһ¶бҹ– бһ”бһҫбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸ Table бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҖбҹ’бһ“бһ»бһ„ pgAdmin бһҗбҹ’бһ„бҹғбһҖбҹ’бһҡбҹ„бһҷбһўбҹ’бһ“бһҖбһ”бҹ’бһҸбһјбһҡбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһ¬бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҖбҹ’бһҡбһ»бһҳ бһҳбһ·бһҸбҹ’бһҸбһ—бһҖбҹ’бһҸбһ·бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҳбһ·бһ“бһҠбһ№бһ„бһҗбһ¶бһўбҹ’бһ“бһҖбһҳбһ¶бһ“ Table бһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮ бһ бһҫбһҷбһҳбһ¶бһ“ Column бһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮбһ‘бҹҒбҹ”
рҹ‘үбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ (Migration): бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠ (JavaScript) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸ Tableбҹ” бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһҗбҹӮбһҳ Column (бһҠбһјбһ…бһҮбһ¶ age бһ¬ grade) бһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ”бһ„бҹ’бһҖбһҫбһҸ File Migration бһҗбҹ’бһҳбһёбҹ”
рҹ‘үбһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбҹ– бһўбҹ’бһ“бһҖбһўбһ¶бһ… "Undo" (Rollback) бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһ‘бһҳбҹ’бһҡбһ„бҹӢ Database бһ…бһ¶бһҹбҹӢбһңбһ·бһүбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ”

****Seeder бһӮбһәбһҮбһ¶бһҖбһ¶бһҡ "бһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбҹҶбһҡбһј" бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ Database бһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бҹ”
рҹ‘үбһ”бһүбҹ’бһ бһ¶бҹ– бһҡбһ¶бһӣбҹӢбһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸ Table бһҗбҹ’бһҳбһё бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһўбһ„бҹ’бһӮбһ»бһҷбһңбһ¶бһҷбһ”бһүбҹ’бһ…бһјбһӣбһҲбҹ’бһҳбҹ„бҹҮбһҹбһ·бһҹбҹ’бһҹбһҳбҹ’бһҸбһ„бһҳбҹ’бһ“бһ¶бһҖбҹӢбҹ—бһҸбһ¶бһҳ Form бһҠбһҫбһҳбҹ’бһ”бһёбһҸбҹҒбһҹбҹ’бһҸбһҖбһјбһҠ бһҠбҹӮбһӣбһңбһ¶бһҒбһ¶бһҸбһ–бҹҒбһӣбһҒбҹ’бһӣбҹҮбҹ”
рҹ‘үбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ (Seeder): бһўбҹ’бһ“бһҖбһҹбһҡбһҹбҹҒбһҡбһҖбһјбһҠбһұбҹ’бһҷбһңбһ¶бһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ·бһҹбҹ’бһҹбһӮбҹҶбһҡбһј бҹЎбҹ  бһ¬ бҹЎбҹ бҹ  бһ“бһ¶бһҖбҹӢбһ—бҹ’бһӣбһ¶бһҳбҹ—бҹ”
рҹ‘үбһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбҹ– бһҮбһҪбһҷбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҸбҹҒбһҹбҹ’бһҸбһҳбһ»бһҒбһ„бһ¶бһҡ Search, Filter бһ¬ Pagination бһ”бһ¶бһ“бһ—бҹ’бһӣбһ¶бһҳбҹ—бһҠбҹ„бһҷбһҳбһ·бһ“бһ”бһ¶бһ…бҹӢбһңбһ¶бһҷбһ”бһүбҹ’бһ…бһјбһӣбһҠбҹ„бһҷбһҠбҹғбҹ”

***бһ”бһ„бҹ’бһҖбһҫбһҸ Model бһ“бһ·бһ„ Migration (Name, Age, Grade)
npx sequelize-cli model:generate --name Student --attributes name:string,age:integer,grade:string

***бһ”бһүбҹ’бһҮбһјбһ“ Table бһ‘бҹ…бһҖбһ¶бһ“бҹӢ PostgreSQL
npx sequelize-cli db:migrate

***бһ”бһ„бҹ’бһҖбһҫбһҸ Seeder (бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹҒбһҹбҹ’бһҸ)
1.бһңбһ¶бһҷбҹ– npx sequelize-cli seed:generate --name demo-student
2.бһ”бһҫбһҖ File бһҗбҹ’бһҳбһёбһҠбҹӮбһӣбһ‘бһҫбһ”бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбҹ’бһ“бһ»бһ„ Folder seeders/ бһҡбһҪбһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбҹҶбһҡбһјбҹ”
3.бһҡбһҪбһ…бһңбһ¶бһҷбҹ– npx sequelize-cli db:seed:all бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбҹ”

seeders/student.js
'use strict';

/** @type {import('sequelize-cli').Migration} */
module.exports = {
  async up (queryInterface, Sequelize) {
    // бһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбҹҶбһҡбһјбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ Table "Students"
    await queryInterface.bulkInsert('Students', [
      {
        name: 'Sok Chea',
        age: 18,
        grade: '12A',
        createdAt: new Date(),
        updatedAt: new Date()
      },
      {
        name: 'Vannak Sing',
        age: 17,
        grade: '11B',
        createdAt: new Date(),
        updatedAt: new Date()
      }
    ], {});
  },

  async down (queryInterface, Sequelize) {
    // бһӣбһ»бһ”бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбҹҶбһҡбһјбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһңбһ·бһү бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ¶бһ“бһҖбһ¶бһҡбһ’бҹ’бһңбһҫ Rollback
    await queryInterface.bulkDelete('Students', null, {});
  }
};
